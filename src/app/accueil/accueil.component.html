Initiation au framework ANGULAR

Mise en place du framework :

# Description :

Angular est un framework Javascript Front-End, basé sur du TypeScript, et développé par Google.

Dernière version au 23/04/2019:

7.2.13 (12/04/2019)

# Installation / mise en place :

Installation de l'environnement nécessaire à l'utilisation d'Angular.

Tapez les lignes de commandes dans une invite de commande en mode administrateur :
(dans barre de recherche de windows, taper 'cmd' puis, faire clic droit et "Exécuter en tant qu'administrateur")

  1. NodeJS : Installer la dernière version LTS
  2. NPM : npm install -g npm@latest
  3. ANGULAR/CLI (Command-Lign Interface) : npm isntall -g @angular/cli@latest

Si l'on souhaite intégrer Bootstrap à son projet angular, après l'installation de angular/cli, taper :

npm install bootstrap@latest --save
(pour la dernière version de bootstrap; sinon taper boostrap@n°VersionSouhaitée)

Ne pas oublier de rajouter, dans angular.json, de rajouter dans la partie 'architecte' :

"styles":["node_modules/bootstrap/dist/css/bootstrap.css",

# Création d'un projet Angular :

Dans un terminal, se placer dans le dossier où l'on souhaite créer son projet et taper :

ng new nom-projet

Pour lancer ensuite un serveur permettant le lancement du projet, taper :

ng serve

ou, pour le lancer directement dans une nouvelle fenêtre :

ng serve -o

# Création de son premier component/composant Angular :

Dans un terminal, se placer dans le dossier où son projet et taper :

ng generate component nomComponent

Cela revient au même que de taper, de manière plus courte :

ng g c nomComponent
