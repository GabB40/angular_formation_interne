# Structure du dossier 'src' :

le dossier 'src' sera celui dans lequel la grande partie du travail se réalisera (hors test et mise en prod)

Nous y retrouvons 'index.html' et la balise permettant l'intégration du component 'app.component.ts' :

<app-root></app-root>


ainsi qu'e'un fichier 'style.css' qui permettra de généraliser du css à tous les components si voulu
(sinon, pour appliquer du css à un component en particulier, écrire le css dans le fichier .css associé au component)
